Normalenform einer Ebene (aus der Parameterform)

  • Die Generelle Normalenform:

    n⃗ · ( x⃗ - p⃗ ) = 0


  • Die Normalenform nutzt das Skalarprodukt um eine Ebene darzustellen.
    da

    jeder Punkt auf der Ebene ja zum Normalvektor orthogonal sein muss

    , muss das skalarprodukt aus diesem und
    dem Vektor eines Punktes gleich 0 sein.

  • n⃗

    ist der Normalvektor

  • p⃗

    ist der "Aufpunkt"-Vektor

  • x⃗

    ist unser beliebiger Vektor/Punkt
